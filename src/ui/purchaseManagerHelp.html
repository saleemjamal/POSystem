‚ùìFrequently Asked Questions

Q: How do I know what quantities to order?

The system automatically recommends quantities based on sales data and current stock levels. You don't need to calculate anything!

Q: Can I create POs for all stores at once?

Yes! Use "üöÄ Generate Multiple POs" to create POs for all stores that need stock. This is the fastest way.

Q: What if I make a mistake in a PO?

You can edit the PO archive file directly, but only before it's sent to the distributor. After sending, contact administrators for changes.

Q: How do I know if my PO was sent?

Check the Status column in POTracking sheet. It will show "Sent" when emailed to the distributor.

Q: Can I see what's been delivered?

Yes! Check the FulfillmentPercentage column to see delivery progress for each PO.
