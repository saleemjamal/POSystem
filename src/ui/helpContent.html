PO System Help

  • Create PO: Fill out the form and click Create Purchase Order. A new PO will be added to the POTracking sheet with Status set to Pending.
  • Approve PO: When ready, check the Approved checkbox in the POTracking sheet. Set Status to Approved (manually).
  • Send PO: Use the menu item Send Approved POs. Only POs with Approved checked and EmailSent unchecked will be sent. After sending, EmailSent will be checked and Status set to Sent.
  • Status Column: Tracks the PO's progress: Pending → Approved → Sent → Received (last step is manual).
  • Help: Click the Help button in the sidebar at any time to view these instructions.

For further assistance, contact your system administrator.
